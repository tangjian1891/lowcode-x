## 🚀 核心项目

### 低代码表单管理系统

一个基于 NestJS + MongoDB + Vue3 的企业级低代码平台

**项目背景**
传统表单开发需要大量重复的CRUD代码，本项目通过可视化设计器和动态渲染引擎，实现表单的快速创建和管理。

**技术架构**

- 前端：Vue3 + TypeScript + Element Plus + Vite
- 后端：NestJS + MongoDB + Mongoose + JWT
- 部署：Docker + Nginx + PM2

**核心功能模块**

- [表单设计器](./form-designer.md) - 可视化拖拽表单构建
- [动态渲染引擎](./form-renderer.md) - 基于Schema的表单渲染
- [数据管理系统](./data-management.md) - 动态表单数据CRUD
- [权限控制](./permission-system.md) - 基于RBAC的权限管理
