## 🎯 实践项目

### 低代码表单系统后端

基于 NestJS + MongoDB 的企业级后端服务

**技术特色：**

- 🏗️ 模块化架构设计
- 🔒 JWT + RBAC 权限体系
- 📊 动态Schema管理
- 🚀 RESTful API 设计

**核心模块：**

- 用户认证与权限管理
- 表单Schema动态生成
- 数据CRUD操作封装
- 文件上传与管理
